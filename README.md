#F

Functional programming utilities for C# & Unity3D

##Features
- Use more lamdas and fewer loops
- Consistent parameter ordering
- Functions are side-effect free
- Easy to transform data

##Supported Data Types
- `object`
- `IDictionary<K, V>`
- `ICollection<T>`
- `ICollection<List<T>>`
- `T[][]`
- `T[,]`
